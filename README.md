# 🎥 GPM CFTV Suite

**Sistema de Monitoramento CFTV - Armazém Paraíba**

---

## 📦 Sobre o Projeto

Suite profissional para gerenciamento e visualização de câmeras IP.

### 🔧 GPM CFTV Studio (Windows)
- Adicionar/Remover câmeras
- Configurar IP, usuário, senha
- Detectar perfil RTSP automaticamente
- Testar conexão
- Gerar Viewer para Linux

### 🖥️ GPM CFTV Viewer (Linux)
- Visualização em tempo real
- Reconexão automática
- Layout adaptativo
- Baixo consumo de CPU/RAM
- AppImage (sem instalação)

---

## 🚀 Tecnologias

- **Python 3.x**
- **PySide6** (Interface gráfica)
- **GStreamer** (Motor RTSP)
- **Docker** (Desenvolvimento)
